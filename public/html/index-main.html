CV NodeJs Web Site

CV NodeJs es un sitio web creado con la firme inteción de volcar los conocimientos adquiridos sobre desarrollo web y a la vez como medio de difusión para que las empresas de software que quieran contratar personal tengan conocimiento de mi curriculum vitae.

CV NodeJs asi como su logo fue inspirado en el entorno NodeJs. Las razones por las que se le eligio son:

  • Permite el uso de Javascript en en el backend.
  • Ejecucion asincronica de las tareas.
  • Baja curva de aprendisaje.

Las otras tecnologías que se estan usando son:

  • HEROKU como hosting web.
  • GITHUB para control de versiones.
  • HTML5, CSS3 y Javascript, para todo el front-end.
  • Angular JS para mejorar el tiempo de carga.
  • NodeJs como servidor web
